# Break The BriX

A Java Swing brick breaker game created as a university project.

## Features

- Player name input
- Difficulty selection: Easy, Medium, Hard
- Score system
- Lives system
- Brick collision
- Paddle control
- Game over and win screen
- Restart option

## Technologies Used

- Java
- Java Swing
- AWT Graphics
- Object-Oriented Programming

## How to Run

Clone the project:

```bash
git clone https://github.com/Ayramuh-ynah/break-the-brix.git

```

Go to the project folder:

````bash
cd break-the-brix




Compile:

```bash
javac *.java
````

Run:

```bash
java Main
```

## Controls

- Left Arrow: move paddle left
- Right Arrow: move paddle right
- Enter: restart after win/game over
- 1, 2, 3: select Easy, Medium, Hard after the game stops
